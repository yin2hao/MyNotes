##  边界符

^：以` `开始
\*：以` `结束

| 边界符 | 说明              |
| --- | --------------- |
| `^` | 表示匹配行首的文本（以谁开始） |
| `$` | 表示匹配行尾的文本（以谁结束） |
如果`^`和`$`在一起，表示必须是精确匹配。


## 量词

| 量词    | 说明         |
| ----- | ---------- |
| *     | 重复零次或更多次   |
| +     | 重复一次或更多次   |
| ?     | 重复零次或一次    |
| {n}   | 重复 n 次     |
| {n,}  | 重复 n 次或更多次 |
| {n,m} | 重复 n 到 m 次 |
