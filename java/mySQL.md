
## 创建
```
CREATE TABLE 表格名(
	列名1 INT AUTO_INCREMENT PRIMARY KEY,
	列名2 VARCHAR() NOT NULL,
	列名3 DATE NULL 
);
```

## 
```
INSERT INTO 数据库名.表格名(
列名1， 列名2， 列名3)
VALUES(
数值1， 数值2， 数值3)
```


## 
```
ALTER TABLE 数据库名.表格名
ADD 列名 数据类型 默认条件
```


## 
```

SET 值
WHERE 条件
```
